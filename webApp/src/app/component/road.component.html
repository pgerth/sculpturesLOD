Road

This is an showcase implementation for the use of TurfJS to overcome the limits of the ElasticSearch geo shape query. In that example, we want to query for all white marble objects near the well known roman road Via appia. TurfJS takes the Line Geometry of the Via Appia and calculates a buffer of 1km (see right). The derived geometry is further used to perform a query with ElasticSearch geo shape query.
